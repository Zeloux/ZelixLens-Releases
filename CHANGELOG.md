# Changelog

This file records customer-facing ZelixLens CS2 Edition releases. Release assets and exact publication times remain available on the [GitHub Releases page](../../releases).

## 2.0.0 - 2026-07-19

- Published the current Windows 10/11 x64 customer package.
- Added the full kernel-backed, read-only transport used by the current runtime.
- Shipped the secure launcher with encrypted local key storage and authenticated automatic updates.
- Included signed update metadata, public SHA-256 checksums, release provenance, and consolidated license notices.
- Expanded player visuals, radar and world information, aim configuration, presets, and display controls.

The public release tag is `v2.0.0`. Earlier `v1.x` identifiers referred to updater revisions rather than the customer-facing product version.

## Release policy

- New customer versions receive a unique semantic-version tag such as `v2.0.1`.
- Published releases are retained to provide a clear update history.
- Compatibility claims are dated in the release notes and are not permanent detection-status guarantees.
