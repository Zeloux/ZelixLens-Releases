# Contributing

ZelixLens is proprietary and this repository intentionally does not contain the product source code. Source-code contributions are not accepted here.

Helpful public contributions include:

- correcting documentation errors
- improving download-verification instructions
- reporting broken links or unclear release notes
- suggesting non-sensitive support documentation

Start with a GitHub Discussion before preparing a documentation pull request. Security issues, access keys, account details, and suspected release tampering must be reported privately through [SECURITY.md](SECURITY.md).

Documentation contributions must be accurate, respectful, narrowly scoped, and free of private customer information. By submitting documentation, you confirm that you have the right to contribute it and permit Zaroomx to publish it with this repository.
